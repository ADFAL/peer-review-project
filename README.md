# ⚙️ How to Run the Project

---

## 🟢 Run backend server
```
uvicorn api.app:app --reload
```
##  URL
```
http://127.0.0.1:8000
```

## 🟢 Frontend (React)
## Go to frontend folder
```
cd frontend
```
## Install dependencies
```
npm install
```
## Run frontend
```
npm run dev
```
## Frontend URL
```
http://localhost:5173
```
## Run streamlit
```
streamlit run streamlit_app/app.py
```

## 🔴 Backend (FastAPI)

### Install dependencies
```bash
pip install -r requirements.txt
```
